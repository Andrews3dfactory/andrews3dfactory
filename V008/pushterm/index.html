PushTerm

PushTerm is a Python command-line tool that modifies G-code for the Bambu Lab X1 Carbon 3D printer. It adds push-off commands so the toolhead ejects prints after a cooldown delay, avoiding extra hardware. Made to simplify print removal and protect your printer.
